URL Shortener Microservice

How to create a shortened URL:

Here we are creating a shortened url for https://example.com

new/https://example.com

You will receive a JSON response with either an error or the shortened URL in the following format:

{"original_url":"https://example.com","short_url":"1"

{"error":"URL invalid"}
